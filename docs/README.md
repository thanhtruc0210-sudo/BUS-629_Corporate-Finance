# docs/

Project documentation for the BUS-629 Apple Inc. (AAPL) financial-analysis capstone. Contains `decisions/` (the Stage 2 company-selection memo and any instructor-feedback responses), `specs/` (the Stage 4 technical specification that drives the LLM analysis), `plans/` (planning scratch), and `templates/` (local copies/pointers for instructor templates). Dated artifacts follow the `YYYY-MM-DD-pham-apple-{kind}.md` naming convention; per-directory READMEs use the fixed name `readme.md`. This directory spans Stages 2 and 4 of the project pipeline. Start with `decisions/` for the company rationale, then `specs/` for the model specification.

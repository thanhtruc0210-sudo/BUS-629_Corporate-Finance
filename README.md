## Project Overview

Comprehensive financial analysis of Apple Inc. using ratio analysis, Du Pont decomposition, and strategic valuation. This project demonstrates spec-driven design methodology applied to financial modeling and LLM-assisted analysis.

**Company:** Apple Inc. (NASDAQ: AAPL)  
**Fiscal Year:** 2025 (Oct 2024 - Sep 2025)  
**Analysis Focus:** Liquidity, Profitability, Leverage, Efficiency, Growth, Valuation

## Repository Structure

- **`docs/`** — Project documentation (decisions, plans, specifications)
- **`models/`** — Financial templates and built models
- **`data/`** — Source financial data and assumptions
- **`analysis/`** — Financial ratio analysis and interpretations
- **`deliverables/`** — Final project outputs

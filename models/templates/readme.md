# Performance Ratios Template

Instructor-provided Excel template defining the standard ratio model architecture.

## Files

- `performance-ratios-template.xlsx` — Standard template (Stage 1)

## Template Structure

**Worksheets:**
- **Cover** — Metadata, sources, reporting standards
- **Balance Sheet** — Assets, liabilities, equity
- **Income Statement** — Revenue, expenses, net income
- **Cash Flow Statement** — Operating, investing, financing activities
- **Ratios** — Calculated financial ratios by category
- **Notes** — Assumptions and calculations

**Named Ranges:** BAL_*, INC_*, CASH_*, RATIO_*

*Do not modify this template file.*

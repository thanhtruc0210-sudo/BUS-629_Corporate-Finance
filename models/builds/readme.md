# Populated Financial Models

Excel workbooks populated with actual company financial data.

## Contents

- Populated templates with Apple Inc. FY2025 financials
- Historical data (FY2024 & FY2025 for comparison)
- Market assumptions (share price, shares outstanding, WACC, tax rate)

## File Naming Convention

`YYYY-MM-DD-{lastname}-{company-slug}-financials.xlsx`

Example: `2026-05-15-pham-apple-financials.xlsx`

## Validation Checklist

- Balance sheet equation: Assets = Liabilities + Equity
- No formula errors (#REF!, #DIV/0!, #NAME?)
- Prior-year data populated
- Ratios tab auto-calculates

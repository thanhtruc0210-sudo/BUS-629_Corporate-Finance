# models/

Excel financial models for the Apple Inc. (AAPL) ratio analysis. `templates/` holds the instructor-provided `performance-ratios-template.xlsx` (Stage 1 architecture reference); `builds/` holds the populated Stage 3 workbook `2026-05-16-pham-apple-financials.xlsx` with the Balance Sheet, Income Statement, Cash Flow, and Ratios tabs filled from Apple's FY2025 10-K. Built workbooks follow the `YYYY-MM-DD-pham-apple-financials.xlsx` naming convention; the upstream template keeps its original instructor filename. This directory covers Stages 1 and 3. The Stage 3 workbook is the single source of truth for every figure in the Stage 4 spec and Stage 5 analysis.
